Header: Commodity Constraints Injection Coefficient

Table Details:
Units:	~
Mode:	Input Only
Multi-band:	False
Default Value:	0
Validation Rule:	Any Value
Key Property:	No
Description:	Coefficient of injection of the Commodity into inventory

Paragraphs:
Commodity Constraints Injection Coefficient sets the coefficient of Injection in the Constraint.

With the coefficient represented by a (i,j) the following
          term is added to the formulation:

... + a (i,j) × Commodity Injection ... {≤,
            ≥, =} b j
