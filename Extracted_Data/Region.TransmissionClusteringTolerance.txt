Header: Region Transmission Clustering Tolerance

Table Details:
Units:	%
Mode:	Input Only
Multi-band:	False
Default Value:	100
Validation Rule:	Between 0 and 100
Key Property:	No
Description:	Cluster nodes until this level of accuracy is reached (100% means no clustering)

Paragraphs:
Region Transmission Clustering Tolerance is a clustering criteria based on the level of accuracy required (where 100% means no clustering).

See the Region Transmission Clustering Level topic for details.

If this option is set to a value less than 100% clustering occurs until this level of accuracy is reached.
