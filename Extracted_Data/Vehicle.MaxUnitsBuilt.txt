Paragraphs:
Vehicle Max Units Built is the key
          property that indicates expansion should be optimized for the Charging
          Station. It is the maximum number of units constructed in LT
            Plan in total over the entire planning horizon. Use the Max
            Units Built in Year property to limit the number of units built
          in any given year, and the Project
            State Date property to set a date after which expansion is
          allowed.
