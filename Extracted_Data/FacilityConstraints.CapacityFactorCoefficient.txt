Header: Facility Constraints Capacity Factor Coefficient

Table Details:
Units:	-
Mode:	Input Only
Multi-band:	False
Default Value:	0
Validation Rule:	Any Value
Key Property:	No
Description:	Coefficient of capacity factor

Paragraphs:
Facility Constraints Capacity Factor Coefficient is the left-hand side coefficient of Facility Capacity Factor in the
          constraint.

With the coefficient represented by a (i,j) the following
          term is added to the formulation:

... + a (i,j) × Facility Capacity Factor ... {≤,
            ≥, =} b j
