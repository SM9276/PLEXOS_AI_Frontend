Header: Gas Field Internal Volume Scalar

Table Details:
Units:	-
Mode:	Input Only
Multi-band:	False
Default Value:	1
Validation Rule:	>0
Key Property:	No
Description:	Storage volume scaling factor used internal to the mathematical program.

Paragraphs:
Gas Field Initial Volume is used in the simulation's mathematical program to scale the gas field volumes. The parameter can be used to help resolve scaling issues when modelling a system that has very large volumes.

NB: If using this parameter be careful to use the same scaling parameter among all interconnected gas objects:
Lists:
	- Gas Storage Internal Volume Scalar
	- Gas Pipeline Internal Volume Scalar
