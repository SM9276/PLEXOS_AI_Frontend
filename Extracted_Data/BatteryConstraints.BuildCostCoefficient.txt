Header: Battery Constraints Build Cost Coefficient

Table Details:
Units:	$
Mode:	Input Only
Multi-band:	False
Default Value:	0
Validation Rule:	Any Value
Key Property:	No
Description:	Coefficient of total build cost

Paragraphs:
Battery Constraints Build Cost is the left-hand side coefficient of Battery Build Cost from the start of the horizon up to the current period in the constraint. The cost is the nominal lump-sum Build Cost i.e without any adjustment for Discount Rate .
