Header: Generator Pump Bid Quantity

Table Details:
Units:	MW
Mode:	Input/Output
Multi-band:	True
Default Value:	0
Validation Rule:	Any Value
Key Property:	No
Description:	Pump load bid quantity in band

Paragraphs:
As an input Generator Pump Bid Quantity is the quantity part of a multi-band bid for Pump Load . The price is set by Pump Bid Price . See the Balancing Markets article for details.

As an output Generator Pump Bid Quantity repeats the input value in incremental style .
