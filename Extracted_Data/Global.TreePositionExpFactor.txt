Global Tree Position Exp Factor
          relates to the Scenario Tree and controls the end positions of each
          stage.

The Tree Stages Position for stage i (from 1 to S in units
          of Tree Period Type ) is
          determined as:

N * ( i /S) ep

where:

ep is the Tree Position
          Exp Factor

S is the number of
          stages, and

N denotes the number of Tree Period Type periods in
          the Horizon
