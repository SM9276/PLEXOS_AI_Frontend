Header: Decision Variable Objective Function Coefficient Day

Table Details:
Units:	-
Mode:	Input Only
Multi-band:	False
Default Value:	0
Validation Rule:	Any Value
Key Property:	Yes
Description:	Objective function value of the generic decision variable in each day

Paragraphs:
Decision Variable Objective Function Coefficient Day is the variant of Objective Function Coefficient that indicates that there is one decision variable for each day of the horizon rather than every interval.
