Key Property:	No
Description:	Minimum production per month

Paragraphs:
Commodity Min Production Month
          creates a Constraint object that
          forces the Production of the
          Commodity each month. By default this is a 'hard' constraint. To
          create a soft constraint, set the Min
            Production Penalty property.
