Description:	Maximum number of synchronous condenser units

Paragraphs:
Generator Sync Cond units sets the number of units that can operate as synchronous condensers for the purposes of providing spinning reserve . For hydro unit is often called tail-water depressed mode. The unit is able to run at notional zero output but provide spinning reserve. You can set the Max Response and Offer Price for unit running in this mode.
