Header: Generator Sync Cond Load

Table Details:
Units:	MW
Mode:	Input/Output
Multi-band:	False
Default Value:	0
Validation Rule:	≥0
Key Property:	No
Description:	Load drawn by a unit in synchronous condenser mode

Paragraphs:
Generator Sync Cond Load is the
          load drawn by a unit running as a synchronous
            condenser to provide spinning reserve and/or inertia. The total
          load drawn is defined as:

Total Sync Cond Load = Units
            Sync Cond × Sync Cond
            Load
