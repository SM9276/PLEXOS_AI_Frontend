Header: Gas Node Max Units Built in Year

Table Details:
Units:	-
Mode:	Input Only
Multi-band:	False
Default Value:	1
Validation Rule:	In (0,1)
Key Property:	No
Description:	Binary indicating if build is allowed in this year

Paragraphs:
Gas Node Max Units Built in Year is a binary (0,1) value indicating if this project is allowed to be built by LT Plan in a given year.  This value should be less than or equal to Max Units Built .
