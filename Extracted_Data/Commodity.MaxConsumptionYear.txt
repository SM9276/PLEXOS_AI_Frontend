Description:	Maximum consumption per year

Paragraphs:
Commodity Max Consumption Year
          creates a Constraint object that
          limits the Consumption of
          the Commodity each year. By default this is a 'hard' constraint. To
          create a soft constraint, set the Max
            Consumption Penalty property.
