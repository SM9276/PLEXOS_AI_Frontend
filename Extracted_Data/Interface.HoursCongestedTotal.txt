Table Details:
Units:	h

Default Value:	
Validation Rule:	
Key Property:	No
Description:	Number of hours interface is congested in the reference direction

Paragraphs:
Interface Hours Congested Total is the number of hours the interface is flowing at either Min Flow or Max Flow and the Shadow Price is non-zero. The units of Hours Congested depends on the number of hours in each simulation interval . Includes hours congested for base case and post-contingent hours.
