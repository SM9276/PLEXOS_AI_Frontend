Description:	Maximum consumption per week

Paragraphs:
Commodity Max Consumption Week
          creates a Constraint object that
          limits the Consumption of
          the Commodity each week. By default this is a 'hard' constraint. To
          create a soft constraint, set the Max
            Consumption Penalty property.
