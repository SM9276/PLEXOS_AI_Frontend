Header: Generator Constraints Committed Capacity Coefficient

Table Details:
Units:	MW
Mode:	Input Only
Multi-band:	False
Default Value:	0
Validation Rule:	Any Value
Key Property:	No
Description:	Coefficient of committed capacity

Paragraphs:
Generator Constraints Committed Capacity Coefficient is the left-hand coefficient on the Dispatchable Capacity in the constraint.

With the coefficient represented by a (i,j) the following
          term is added to the formulation:

... + a (i,j) × Generator ( Dispatchable
            Capacity ) ... {≤, ≥, =} b j
