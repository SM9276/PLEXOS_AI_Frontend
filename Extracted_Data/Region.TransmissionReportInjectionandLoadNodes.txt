Header: Region Transmission Report Injection and Load Nodes

Table Details:
Units:	Yes/No
Mode:	Input Only
Multi-band:	False
Default Value:	False (0)
Validation Rule:	False (0) or True (-1)
Key Property:	No
Description:	If all injection and load buses (nodes) are reported on (regardless of voltage) in this region.

Paragraphs:
Region Transmission Report Injection and Load Nodes turns on reporting of all buses that are either injection points (have generators or other generation sources connected), or load points (have non-zero Load Participation Factor ) regardless of their voltage level and setting of Region Transmission Report Voltage Threshold . It will over write the general Transmission Report Injection and Load Nodes setting.
