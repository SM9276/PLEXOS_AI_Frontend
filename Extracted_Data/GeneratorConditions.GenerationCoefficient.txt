Header: Generator Conditions Generation Coefficient

Table Details:
Units:	MW
Mode:	Input Only
Multi-band:	False
Default Value:	0
Validation Rule:	Any Value
Key Property:	No
Description:	Coefficient of generation in condition

Paragraphs:
This is the coefficient of Generator Generation in the Variable condition equation.

With the coefficient represented by a (i,j) the following
                term is added to the left-hand side of the condition equation:

… + a (i,j) × Generator Generation … {<,≤,
                    ≥,>, =} Profile
