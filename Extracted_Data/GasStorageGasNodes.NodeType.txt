Header: Gas Storage Gas Nodes Node Type

Table Details:
Units:	-
Mode:	Input Only
Multi-band:	False
Default Value:	0
Validation Rule:	In (0,1,2)
Key Property:	No
Description:	Using this input property, a Gas Node defined in the Gas Node collection of a Gas Storage can act as injection only, withdrawal only or both.

Paragraphs:
Default is to allow both injections and withdrawals on a Gas Node. This property can be used if finer control is desired for the Gas Storage operation.
