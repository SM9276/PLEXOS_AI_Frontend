Description:	Maximum amount of fuel input to generator

Paragraphs:
Generator Fuels Max Input sets an upper limit on the amount of the fuel that can be input to the generator. This property is most commonly used to model fuel input from a duct burner in a combined-cycle generator .

Note that for multi-unit generators this is a maximum total input across all units, not a per-unit value.
