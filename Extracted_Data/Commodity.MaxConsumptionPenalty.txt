Header: Commodity Max Consumption Penalty

Table Details:
Units:	$/~
Mode:	Input Only
Multi-band:	False
Default Value:	-1
Validation Rule:	Any Value
Key Property:	No
Description:	Penalty for violation of [Max Consumption] constraints

Paragraphs:
Commodity Max Consumption Penalty
          sets the penalty for violating the Max
            Consumption constraint and its period variants. The default
          value of -1 means 'hard constraint'.
