Header: Facility Constraints Units Operating Coefficient

Table Details:
Units:	-
Mode:	Input Only
Multi-band:	False
Default Value:	0
Validation Rule:	Any Value
Key Property:	No
Description:	Coefficient on the number of units operating

Paragraphs:
Facility Constraints Units Operating Coefficient is the left-hand side term of the number
          of Units Operating at the
          facility in the constraint.

With the coefficient represented by a (i,j) the following
          term is added to the formulation:

... + a (i,j) × Facility Units Operating ... {≤,
            ≥, =} b j
