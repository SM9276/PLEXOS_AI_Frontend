Generator Fixed Load Global
          controls how the Fixed Load property is interpreted for multi-unit facilities. It can take the
          following values:

The values are for the total Generation across all units , except in LT Plan for expansion
              options in which case Fixed
                Load is the value for each existing unit or unit built.
