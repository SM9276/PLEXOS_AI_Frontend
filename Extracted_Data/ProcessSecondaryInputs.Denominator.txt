Header: Process Secondary Inputs Denominator

Table Details:
Units:	-
Mode:	Input Only
Multi-band:	False
Default Value:	0
Validation Rule:	In (0,1)
Key Property:	No
Description:	The denominator for the Ratio property where "Input" refers to the Primary Input and "Output" refers to the Primary Output

Paragraphs:
Process Secondary
            Inputs Denominator relates to the Ratio property. It can take the following values:

The denominator of Ratio is the Primary Input Consumption i.e.

Ratio = Secondary
                Inputs Consumption / Primary
                Input Consumption

The denominator of Ratio is the Primary Output Production i.e.

Ratio = Secondary
                Inputs Consumption / Primary
                Output Production
