Description:	Minimum production over custom timeframe

Paragraphs:
Commodity Min Production Custom
          creates a Constraint object that
          forces the Production of the
          Commodity over a custom timeframe. By default this is a 'hard'
          constraint. To create a soft constraint, set the Min
            Production Penalty property.
