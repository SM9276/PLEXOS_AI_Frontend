Header: Commodity Constraints Consumption Coefficient

Table Details:
Units:	~
Mode:	Input Only
Multi-band:	False
Default Value:	0
Validation Rule:	Any Value
Key Property:	No
Description:	Coefficient of Commodity consumption in the Constraint

Paragraphs:
Commodity Constraints Consumption Coefficient sets the coefficient of Consumption in the Constraint .

With the coefficient represented by a (i,j) the following
          term is added to the formulation:

... + a (i,j) × Commodity Consumption ... {≤,
            ≥, =} b j
