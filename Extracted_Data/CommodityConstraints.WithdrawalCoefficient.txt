Header: Commodity Constraints Withdrawal Coefficient

Table Details:
Units:	~
Mode:	Input Only
Multi-band:	False
Default Value:	0
Validation Rule:	Any Value
Key Property:	No
Description:	Coefficient of withdrawal of the Commodity from inventory

Paragraphs:
Commodity Constraints Withdrawal Coefficient sets the coefficient of Withdrawal in the Constraint.

With the coefficient represented by a (i,j) the following
          term is added to the formulation:

... + a (i,j) × Commodity Withdrawal ... {≤,
            ≥, =} b j
