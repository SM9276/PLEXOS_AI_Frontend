Key Property:	No
Description:	Minimum production per year

Paragraphs:
Commodity Min Production Year
          creates a Constraint object that
          forces the Production of the
          Commodity each year. By default this is a 'hard' constraint. To create
          a soft constraint, set the Min
            Production Penalty property.
