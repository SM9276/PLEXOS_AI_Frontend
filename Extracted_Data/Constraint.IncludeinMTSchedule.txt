Header: Constraint Include in MT Schedule

Table Details:
Units:	Yes/No
Mode:	Input Only
Multi-band:	False
Default Value:	True (-1)
Validation Rule:	False (0) or True (-1)
Key Property:	No
Description:	If the constraint is modelled in the MT Schedule phase.

Paragraphs:
Constraint Include in MT Schedule switches on/off enforcing the constraint in the MT Schedule simulation phase. The flag can take these values:

Note that constraints defined with RHS Custom are automatically excluded unless they have been decomposed by the higher level LT Plan phase.The value of this property cannot be changed during the simulation itself, i.e. it cannot be used to toggle the constraint in and out of the specific simulation phase.
