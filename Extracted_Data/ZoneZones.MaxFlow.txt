Header: Zone Zones Max Flow

Table Details:
Units:	MW
Mode:	Input Only
Multi-band:	False
Default Value:	1E+30
Validation Rule:	Any Value
Key Property:	No
Description:	Maximum flow allowed between the zones

Paragraphs:
Zone Zones Max Flow is Maximum flow allowed between the zones.

Zone Zones Max Flow A->B = ∑ Export Line Flow A->B - ∑
          Import Line Flow B->A

The max flow constraint is always formulated upfront.
