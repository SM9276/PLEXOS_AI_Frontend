This is the priority value for the default objective function. This
          is used for sequential Lexicographic multi-objective. Solution 1 is
          the highest priority and the order of priority is 2,3,... and so on. The
          default cost objective has highest priority by default. The priority
          of user-defined objective function is set with the Objective Priority property.
