Description:	Minimum consumption per week

Paragraphs:
Commodity Min Consumption Week
          creates a Constraint object that
          forces the Consumption of
          the Commodity each week. By default this is a 'hard' constraint. To
          create a soft constraint, set the Min
            Consumption Penalty property.
