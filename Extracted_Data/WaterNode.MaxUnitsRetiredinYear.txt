Header: Water Node Max Units Retired in Year

Table Details:
Units:	-
Mode:	Input Only
Multi-band:	False
Default Value:	1
Validation Rule:	In (0,1)
Key Property:	No
Description:	Maximum number of units automatically retired in any single year of the planning horizon

Paragraphs:
Water Node Max Units Retired in Year is a binary (0,1) value indicating if this resource is allowed to be retired by LT Plan in a given year.  This value should be less than or equal to Max Units Retired .
