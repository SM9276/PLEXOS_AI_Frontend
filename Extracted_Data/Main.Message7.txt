Header: Message 7

Table Details:
Severity:	Serious (value = 2)
Default Action:	Warn (value = 1)
Description:	Optimizer returned the status: <value>

Paragraphs:
Message 7 is generated by the selected mathematical programming solver selected via the Performance SOLVER setting.

The message is raised whenever a solution status is returned from the solver that is not recognized by the simulator. For example some solvers can return a status of "UNKNOWN" when numerical stability or other issues caused the solver to finish with a solution that cannot be proven optimal.

This warning should not be ignored. Please contact support for further investigation.
