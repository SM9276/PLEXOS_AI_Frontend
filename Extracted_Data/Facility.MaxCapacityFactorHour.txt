Header: Facility Max Capacity Factor Hour

Table Details:
Units:	%
Mode:	Input Only
Multi-band:	False
Default Value:	100
Validation Rule:	Between 0 And 100
Key Property:	No
Description:	Maximum capacity factor in hour

Paragraphs:
Facility Max Capacity Factor Hour
          sets an upper bound on the facility's Capacity
            Factor in each hour of the simulation. This variant only applies
          to sub-hourly models i.e. where Horizon Periods per Day is greater
          than 24. See Max Capacity
            Factor for more details.
