Header: Commodity Constraints Capacity Built Coefficient

Table Details:
Units:	-
Mode:	Input Only
Multi-band:	False
Default Value:	0
Validation Rule:	Any Value
Key Property:	No
Description:	Coefficient of capacity built

Paragraphs:
Commodity Constraints Capacity Built Coefficient is the left-hand side coefficient of Commodity Capacity Built in the
          constraint.

With the coefficient represented by a (i,j) the following
          term is added to the formulation:

... + a (i,j) × Commodity Capacity Built ... {≤,
            ≥, =} b j
