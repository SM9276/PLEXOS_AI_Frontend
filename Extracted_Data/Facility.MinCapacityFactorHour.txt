Header: Facility Min Capacity Factor Hour

Table Details:
Units:	%
Mode:	Input Only
Multi-band:	False
Default Value:	0
Validation Rule:	Between 0 And 100
Key Property:	No
Description:	Minimum capacity factor in hour

Paragraphs:
Facility Min Capacity Factor Hour
          sets a lower bound on the facility's Capacity
            Factor in each hour of the simulation. This variant only applies
          to sub-hourly models i.e. where Horizon Periods per Day is greater
          than 24. See Min Capacity
            Factor for more details.
