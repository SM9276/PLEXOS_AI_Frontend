Paragraphs:
Line Max Units Built in Year flag if LT Plan is allowed to build the Line in a given year of the planning horizon. This property should be used in combination with Max Units Built and offers a finer control over the timing of new builds.

Example

In this example the Line can be built any time from 2012 onwards, but not in the years 2018-20.

See also: Line Min Units Built in Year .

Note: The Line Units property must be set to zero, as shown in the example above, if using this property.
