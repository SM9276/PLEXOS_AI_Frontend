Key Property:	No
Description:	If [Min Load] applies across all units or unit-by-unit

Paragraphs:
Generator Min Load Global
          controls how the Min Load property is interpreted for multi-unit facilities. It can take the
          following values:

The values are for the total Generation across all units , except in LT Plan for expansion
              options in which case Min Load is the value for each existing unit or unit built.
