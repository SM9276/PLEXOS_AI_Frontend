Key Property:	No
Description:	Maximum consumption per day

Paragraphs:
Commodity Max Consumption Day
          creates a Constraint object that
          limits the Consumption of
          the Commodity each day. By default this is a 'hard' constraint. To
          create a soft constraint, set the Max
            Consumption Penalty property.
