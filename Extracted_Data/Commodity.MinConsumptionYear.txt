Description:	Minimum consumption per year

Paragraphs:
Commodity Min Consumption Year
          creates a Constraint object that
          forces the Consumption of
          the Commodity each year. By default this is a 'hard' constraint. To
          create a soft constraint, set the Min
            Consumption Penalty property.
