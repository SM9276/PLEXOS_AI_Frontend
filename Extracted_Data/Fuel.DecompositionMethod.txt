Header: Fuel Decomposition Method

Table Details:
Units:	-
Mode:	Input Only
Multi-band:	False
Default Value:	1
Validation Rule:	In (0,1,2)
Key Property:	No
Description:	Method used to pass the optimal stockpile trajectory from one simulation phase to the next.

Paragraphs:
Fuel Decomposition Method controls the way the simulator decomposes the optimal LT Plan or MT Schedule sequence of Closing Inventory values in time down to ST Schedule .

Please refer to the Storage Decomposition Method for a discussion of this property.
