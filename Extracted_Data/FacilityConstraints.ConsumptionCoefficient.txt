Header: Facility Constraints Consumption Coefficient

Table Details:
Units:	-
Mode:	Input Only
Multi-band:	False
Default Value:	0
Validation Rule:	Any Value
Key Property:	No
Description:	Coefficient of Facility consumption in the Constraint

Paragraphs:
Facility Constraints Consumption Coefficient  is the left-hand side coefficient of the
          facility's Consumption in the
          constraint.

With the coefficient represented by a (i,j) the following
          term is added to the formulation:

... + a (i,j) × Facility Consumption ... {≤,
            ≥, =} b j
