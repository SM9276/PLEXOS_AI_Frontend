Key Property:	No
Description:	Maximum production over custom timeframe

Paragraphs:
Commodity Max Consumption Custom
          creates a Constraint object that
          limits the Consumption of
          the Commodity over a custom timeframe. By default this is a 'hard'
          constraint. To create a soft constraint, set the Max
            Consumption Penalty property.
