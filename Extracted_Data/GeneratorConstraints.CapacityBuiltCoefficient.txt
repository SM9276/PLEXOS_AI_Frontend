Header: Generator Constraints Capacity Built Coefficient

Table Details:
Units:	MW
Mode:	Input Only
Multi-band:	False
Default Value:	0
Validation Rule:	Any Value
Key Property:	No
Description:	Coefficient of capacity built

Paragraphs:
Generator Constraints Capacity Built Coefficient is the left-hand side coefficient of Generator Capacity Built in the
          constraint.

With the coefficient represented by a (i,j) the following
          term is added to the formulation:

... + a (i,j) × Generator Capacity Built ... {≤,
            ≥, =} b j
