Header: Flow Path Max Flow Hour

Table Details:
Units:	~
Mode:	Input Only
Multi-band:	False
Default Value:	1E+30
Validation Rule:	Any Value
Key Property:	No
Description:	Maximum flow on the Flow Path in any hour

Paragraphs:
Flow Path Max Flow Hour sets an
          upper bound on the Flow in each hour
          of the simulation. This variant only applies to sub-hourly models i.e. where Horizon Periods
            per Day is greater than 24. See Max
            Flow for more details.
