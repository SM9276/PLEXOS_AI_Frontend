Header: Facility Constraints Units Started Coefficient

Table Details:
Units:	-
Mode:	Input Only
Multi-band:	False
Default Value:	0
Validation Rule:	Any Value
Key Property:	No
Description:	Coefficient on the number of units started

Paragraphs:
Facility Constraints Units Started Coefficient is the left-hand side coefficient on the
          number of Facility Units
            Started in the constraint.

With the coefficient represented by a (i,j) the following
          term is added to the formulation:

... + a (i,j) × Facility Units Started ... {≤,
            ≥, =} b j
