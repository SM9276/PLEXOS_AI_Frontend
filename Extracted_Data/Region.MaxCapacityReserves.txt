Header: Region Max Capacity Reserves

Table Details:
Units:	MW
Mode:	Input/Output
Multi-band:	False
Default Value:	1E+30
Validation Rule:	Any Value
Key Property:	No
Description:	Maximum capacity reserves allowed

Paragraphs:
Region Max Capacity Reserve sets an upper bound on the Capacity Reserves in the Region in LT Plan . The expansion solution will attempt to limit the Capacity Reserves below this value by restricting new builds and/or retiring plant. By default the constraint is 'hard' meaning that it cannot be violated, but the limit can be made 'soft' by setting  the Capacity Excess Price .
