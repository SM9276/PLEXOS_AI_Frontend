Header: Generator Fixed Load Method

Table Details:
Units:	-
Mode:	Input Only
Multi-band:	False
Default Value:	1
Validation Rule:	In (0,1)
Key Property:	No
Description:	Method of interpreting zero values of the [Fixed Load] property.

Paragraphs:
Generator Fixed Load Method controls how the Fixed Load property is handled when it has a zero value:

By setting this to "Relax when Zero" you can use the fixed load constraint sporadically during the horizon e.g. to model a period of generator testing where the generator load follows a fixed pattern. This is consistent with the definition of Fixed Load in the Australian NEM.
