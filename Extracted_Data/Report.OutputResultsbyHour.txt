Header: Report Output Results by Hour

Table Details:
Units:	Yes/No
Default Value:	False (0)
Validation Rule:	False (0) or True (-1)
Description:	If summary results are written by hour

Paragraphs:
Report Output Results by Hour switches on/off writing of summary results for each hour of the simulation horizon and only applies when the simulation is sub-hourly ( e.g. 5-minute) as defined by the Horizon Periods per Day setting.
