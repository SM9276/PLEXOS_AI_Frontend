Header: Line Loss Incr2

Table Details:
Units:	-
Mode:	Input Only
Multi-band:	False
Default Value:	0
Validation Rule:	Any Value
Key Property:	No
Description:	Interconnector loss function quadratic parameter for reference direction flows

Paragraphs:
Line Loss Incr2 is the quadratic (second-order) term in the equation that defines losses for flows in the reference direction i.e. for notionally positive values of Flow . See Line Loss Incr for details and examples.

This value applies in the counter-reference direction if Loss Incr 2 Back is not defined.
