Description:	Maximum number of renominations allowed per renomination window for Base Gas Contracts

Paragraphs:
Gas Contract Max Renomination defines the maximum number of renominations allowed per renomination window for a Base Gas Contract. Renomination windows are determined based on the Gas Contract Renomination Start Period.
