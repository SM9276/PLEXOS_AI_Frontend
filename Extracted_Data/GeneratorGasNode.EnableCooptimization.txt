Header: Generator Gas Node Enable Co-optimization

Table Details:
Units:	Yes/No
Mode:	Input Only
Multi-band:	False
Default Value:	True (-1)
Validation Rule:	False (0) or True (-1)
Key Property:	No
Description:	If the gas node is available for use by the generator

Paragraphs:
Generator Gas Node Enable Co-optimization indicates whether 
          the gas node is available for use by the generator. It can be used to turn off the co-optimization of gas and power. Note that only 
          the first period's value will be used.
