Header: Region Include in Marginal Unit

Table Details:
Units:	Yes/No
Mode:	Input Only
Multi-band:	False
Default Value:	True (-1)
Validation Rule:	False (0) or True (-1)
Key Property:	No
Description:	Flag if the region is included in the Region Marginal Unit Diagnostic

Paragraphs:
Region Include in Marginal Unit is flag that controls when the Region is included in the Diagnostic Region Marginal Unit.

By default all regions are included in the diagnostic. You might want to exclude some regions if the diagnostic information is not required for that region.
