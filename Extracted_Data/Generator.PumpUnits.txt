Paragraphs:
Generator Pump Unit is the number
          of Units that can run in pump mode
          at the pumped storage generator .
          Pump Units defaults to the total number of installed units if not
          defined.

You can restrict the number of units that are allowed to pump at any
          one time with the property Max
            Units Pumping . Pumping can be forced using the property Must
            Pump Units .
