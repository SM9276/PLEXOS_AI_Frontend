Header: Line Loss Incr2 Back

Table Details:

Multi-band:	False
Default Value:	0
Validation Rule:	Any Value
Key Property:	No
Description:	Interconnector loss function quadratic parameter for counter-reference direction flows

Paragraphs:
Line Loss Incr2 Back is the quadratic (second-order) term in the equation that defines losses for flows in the counter-reference direction i.e. for notionally negative values of Flow . See Line Loss Incr for details and examples.

This value defaults to Loss Incr2 if that value is defined.
