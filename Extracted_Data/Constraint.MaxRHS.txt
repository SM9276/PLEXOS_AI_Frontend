Header: Constraint Max RHS

Table Details:
Units:	-
Mode:	Input Only
Multi-band:	False
Default Value:	1E+30
Validation Rule:	Any Value
Key Property:	No
Description:	Maximum allowed value when RHS is calculated dynamically

Paragraphs:
Constraint Max RHS sets an upper limit on the right-hand-side constant of the constraint in any simulation period. This limit applies only to constraints whose right-hand-side values are dynamically calculated based on Variable objects applied to the constraint with RHS Coefficient .
