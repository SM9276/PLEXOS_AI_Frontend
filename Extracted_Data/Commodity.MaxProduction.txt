Key Property:	No
Description:	Maximum production per interval

Paragraphs:
Commodity Max
          Production creates a Constraint object that limits the Production of the Commodity each interval. By default this is a 'hard'
          constraint. To create a soft constraint, set the Max
            Production Penalty property.
