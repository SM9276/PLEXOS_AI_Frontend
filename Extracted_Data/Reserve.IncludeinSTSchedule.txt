Header: Reserve Include in ST Schedule

Table Details:
Units:	Yes/No
Mode:	Input Only
Multi-band:	False
Default Value:	True (-1)
Validation Rule:	False (0) or True (-1)
Key Property:	No
Description:	If the reserve is modelled in the ST Schedule phase.

Paragraphs:
Reserve Include in ST Schedule toggles on/off the reserve in the ST Schedule simulation phase. This flag works in combination with the Is Enabled setting.
