Header: Water Node Min Units Retired

Table Details:
Units:	-
Mode:	Input Only
Multi-band:	False
Default Value:	0
Validation Rule:	In (0,1)
Key Property:	No
Description:	Minimum number of units automatically retired in aggregate over the planning horizon

Paragraphs:
Water Node Min Units Retired is a binary (0,1) value that indicates if the project must be retired inside the planning horizon or not. Note that it is sufficient to set this flag without setting Max Units Retired .
