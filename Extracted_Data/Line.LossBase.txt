Header: Line Loss Base

Table Details:
Units:	-
Mode:	Input Only
Multi-band:	False
Default Value:	0
Validation Rule:	Any Value
Key Property:	No
Description:	Interconnector loss function constant parameter for reference direction flows

Paragraphs:
Line Loss Base is the constant term in the equation that defines losses for flows in the reference direction i.e. for notionally positive values of Flow . See Line Loss Incr for details and examples.

This value applies in the counter-reference direction if Loss Base Back is not defined.
