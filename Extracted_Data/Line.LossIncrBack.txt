Header: Line Loss Incr Back

Table Details:
Units:	-
Mode:	Input Only
Multi-band:	True
Default Value:	0
Validation Rule:	Any Value
Key Property:	No
Description:	Interconnector loss function linear parameter for counter-reference direction flows

Paragraphs:
Line Loss Incr Back is the linear (first-order) term in the equation that defines losses for flows in the counter-reference direction i.e. for notionally negative values of Flow .  See Line Loss Incr for details and examples.

This value defaults to Loss Incr x -1 if that value is defined.
