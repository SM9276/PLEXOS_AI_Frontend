Header: Entity Constraints Production Coefficient

Table Details:
Units:	-
Mode:	Input Only
Multi-band:	False
Default Value:	0
Validation Rule:	Any Value
Key Property:	No
Description:	Coefficient of production from facilities associated with the entity

Paragraphs:
Entity Constraints Production Coefficient is the left-hand side coefficient of the Production in the constraint.

With the coefficient represented by a (i,j) the following
          term is added to the formulation:

... + a (i,j) × Entity Production ... {≤,
            ≥, =} b j
