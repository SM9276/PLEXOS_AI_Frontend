Header: Water Node Min Units Retired in Year

Table Details:
Units:	-
Mode:	Input Only
Multi-band:	False
Default Value:	0
Validation Rule:	In (0,1)
Key Property:	No
Description:	Minimum number of units automatically retired in any single year of the planning horizon

Paragraphs:
Water Node Min Units Retired in Year is a binary (0,1) flag indicating if the resource must be retired in the specified year. This value can logically only be set for one year.
