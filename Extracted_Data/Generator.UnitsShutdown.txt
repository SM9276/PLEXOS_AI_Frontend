Header: Generator Units Shutdown

Table Details:
Units:	-
Mode:	Output Only
Multi-band:	False
Default Value:	
Validation Rule:	
Key Property:	No
Description:	Number of units shutdown

Paragraphs:
Generator Units Shutdown is the
          number of units that stopped operating in the period, and is defined
          as:

Units Shutdown t =
          MAX(0, Units Generating ( t -1) - Units Generating ( t ) )

where: t indexes the time interval;
            and Units
              Generating 0 is defined by Initial
              Units Generating

In summary reporting this is the total number of unit shut downs that
          occurred in the period.
