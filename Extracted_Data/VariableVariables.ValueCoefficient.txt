Description:	Coefficient of the Variable Value in the definition of this Variable

Paragraphs:
Variable Variables Value Coefficient is the left-hand side coefficient on the child
          Variable's Value in the equation
          that defines when the parent conditional Variable is active. This allows you to decide when one Variable is active
          depending on the value of another.
