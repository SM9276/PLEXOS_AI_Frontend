Header: Node Units

Table Details:
Units:	-
Mode:	Input Only
Multi-band:	False
Default Value:	1
Validation Rule:	In (0,1)
Key Property:	Yes
Description:	Flag if bus is in service

Paragraphs:
Node Units can take the values 0 (= out of service) or 1 (= in service). This property must be static within any one simulation, but buses can be moved in and out of service in an entire simulation using scenarios.
