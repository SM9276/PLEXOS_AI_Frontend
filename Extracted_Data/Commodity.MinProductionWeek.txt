Header: Commodity Min Production Week

Table Details:
Units:	~
Mode:	Input Only
Multi-band:	False
Default Value:	0
Validation Rule:	Any Value
Key Property:	No
Description:	Minimum production per week

Paragraphs:
Commodity Min Production Week
          creates a Constraint object that
          forces the Production of the
          Commodity each week. By default this is a 'hard' constraint. To create
          a soft constraint, set the Min
            Production Penalty property.
