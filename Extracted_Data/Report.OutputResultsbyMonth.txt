Header: Report Output Results by Month

Table Details:
Units:	Yes/No
Default Value:	False (0)
Validation Rule:	False (0) or True (-1)
Description:	If summary results are written by month

Paragraphs:
Report Output Results by Month switches on/off writing of summary results for each calendar month of the simulation horizon.
