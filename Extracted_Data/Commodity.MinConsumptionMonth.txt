Description:	Minimum consumption per month

Paragraphs:
Commodity Min Consumption Month
          creates a Constraint object that
          forces the Consumption of
          the Commodity each month. By default this is a 'hard' constraint. To
          create a soft constraint, set the Min
            Consumption Penalty property.
