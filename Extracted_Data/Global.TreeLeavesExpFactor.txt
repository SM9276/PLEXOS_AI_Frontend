Header: Global Tree Leaves Exp Factor

Table Details:
Units:	-
Mode:	Input Only
Multi-band:	False
Default Value:	1
Validation Rule:	≥0
Key Property:	No
Description:	Scenario Tree: Controls the number of samples in each stage

Paragraphs:
Global Tree Leaves Exp Factor
          relates to the Scenario Tree and controls the number of samples in
          each stage.

Tree Stages Leaves (number of samples) for stage i (from 1
          to S) is determined as:

M × ( i /S) el

where:

el is the Tree Leaves
          Exp Factor

S is the number of
          stages, and

M denotes the number of
          samples
