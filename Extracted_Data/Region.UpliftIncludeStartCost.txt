Header: Region Uplift Include Start Cost

Table Details:
Units:	Yes/No
Mode:	Input Only
Multi-band:	False
Default Value:	True (-1)
Validation Rule:	False (0) or True (-1)
Key Property:	No
Description:	If the uplift calculation should include recovery of start costs

Paragraphs:
Region Uplift Include Start Cost toggles generator start-up and shutdown cost in/out of the cost included when calculating uplift .
