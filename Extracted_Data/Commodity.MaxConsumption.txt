Header: Commodity Max Consumption

Table Details:
Units:	~
Mode:	Input Only
Multi-band:	False
Default Value:	1E+30
Validation Rule:	Any Value
Key Property:	No
Description:	Maximum consumption per interval

Paragraphs:
Commodity Max Consumption creates
          a Constraint object that limits
          the Consumption of the
          Commodity each interval. By default this is a 'hard' constraint. To
          create a soft constraint, set the Max
            Consumption Penalty property.
