Description:	Minimum consumption per day

Paragraphs:
Commodity Min Consumption Day
          creates a Constraint object that
          forces the Consumption of
          the Commodity each day. By default this is a 'hard' constraint. To
          create a soft constraint, set the Min
            Consumption Penalty property.
